# ytsMovies
Movie app show all movies from yts website , trailer for each movie , details and screenshots for every movie and can download movie with different qualities.

Libraries :
- RxJava
- RxAndroid
- RxBinding
- RxPermissions
- ButterKnife
- Retrofit
- gson
- okhttp
- okio
- Timber
- Glide
- GlidePalette
- MaterialDialogs
- Calligraphy
